These examples show simple, basic tasks done with the Core framework only (not using DockFrontend).
  1. Hello World: a very simple application
  2. The Stations: shows all the available stations
  3. SplitDockStation: how to define the location and size of Dockables when using a SplitDockStation
  4. Themes: shows all the available themes
  5. Persistent Layout: how to store the location and size of Dockables in a file
  6. Actions: how to use the title of a Dockable as toolbar
